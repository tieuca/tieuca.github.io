== Installation ==
___ From within WordPress ___
1. Visit 'Plugins > Add New'
2. Search for 'AZ Settings'
3. Activate 'AZ Settings' from your Plugins page.


___ Manually ___
1. Upload the foxtool folder to the "/wp-content/plugins/" directory
2. Activate the WP Settings plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
